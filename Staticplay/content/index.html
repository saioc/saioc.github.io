A） 重要注意事项请务必阅读本文！- 顶部

请注意，终极视频播放器的安装和配置并不复杂，但由于它具有许多可自定义的设置，因此可能看起来很复杂，请在尝试将其安装到您自己的页面之前仔细阅读整个文档。基本上必须做的是从我提供的示例中复制一些 html 和 js 代码并将其粘贴到您自己的 html 页面中，当然还要添加您自己的视频文件。

终极视频播放器只需要mp4视频格式，它可以在台式机或移动设备上的所有浏览器中工作，这一点非常重要，因为无需为不支持mp4的浏览器创建多种视频格式，另请注意，MP4视频格式被所有主要浏览器分销商推荐和使用，如谷歌， 微软，苹果，Mozzila等！

默认情况下，Vimeo 视频是禁用的，因为它们不在本地工作，Vimeo API 无法在本地工作，因此必须禁用它才能设置播放器并在本地进行测试，以启用它设置使用Vimeo：“是”

使用 Flash 播放器下的 Safari 或 MAC 下的 Firefox，因为此浏览器不支持 mp4 视频，如果您的视频在完全加载之前没有开始，那么它们最后会用元数据进行编码，这可以通过使用此工具轻松修复。随着此浏览器的改进，这可能会在此浏览器的未来版本中发生变化。

服务器区分字符大小写，因此请确保终极视频播放器设置与提供的示例相同。

使用服务器上具有相对路径的mp4视频文件时，您必须在内容文件夹中添加包含mp4视频的所有文件夹，这对于所有功能正常工作非常重要。

某些功能将无法在本地运行，例如共享按钮或在某些浏览器（如IE和Opera）上播放youtube视频，这是正常行为，因为大多数浏览器不允许这样做，一旦上传到服务器上，它将正常运行。

当终极视频播放器操作出现问题时，终极视频播放器上会出现一个带有问题描述的红色信息框，请注意，此框是为了帮助您了解问题，例如，如果未找到mp4视频文件，则会出现该框并通知您有关问题的信息。单击或触摸时可以关闭该框。

终极视频播放器皮肤由.png图像构成，只有少数更改颜色的设置是可能的，例如更改时间颜色，缩略图文本颜色等。如果您想要不同的外观，则必须修改.png文件。我们的团队中有一位出色的设计师，因此如果您需要定制皮肤，请与我们联系！

    获取总时间

    		 getTotalTime（格式：字符串）：字符串 返回视频总时间/持续时间。format 参数可以是文本 （hh：mm：ss） 或秒。

    go全屏

    		 goFullScreen（）：void 将视频播放器设置为全屏。

    玩

    		 play（）：void 开始/播放播放。

    播放视频

    		 playVideo（videoId）：void 播放基于videoId的视频，计数从0开始。

    玩下一个

    		 播放下一个（）：void 开始/播放下一个视频。

    播放上一页

    		 playPrev（）：void 开始/播放以前的视频。

    玩随机播放

    		 playShuffle（）：void 开始/播放随机播放视频。

    暂停

    		 暂停（）：无效 暂停播放。

    		 setVideoSource（videoSource：String）：void 设置视频源路径或 YouTube 视频 ID。请阅读设置视频、广告（前贴片/中贴片/后贴片）、字幕、hls、360 度视频和海报来源。
